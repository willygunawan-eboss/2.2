# ERP Setup Center

## Overview
Bootstrap wizard sekali pakai telah dihapus. Konfigurasi organisasi, HR, Finance, dan modul pendukung lainnya sekarang terpusat di **ERP Setup Center**. 

## Fitur Utama
1. **Permanent Availability**: Dapat diakses kapan saja melalui Dashboard ("ERP Setup Center").
2. **Module Readiness**: Menampilkan status kesiapan (Ready / In Progress) dan detail jumlah data (Misal: jumlah department, employee).
3. **Smart Resume**: Menyarankan langkah selanjutnya yang perlu diselesaikan melalui tombol "Continue Setup".
4. **Dependencies Control**: Mencegah setup modul jika dependensinya belum siap (Misal: tidak bisa configure Employee jika Organization belum siap).
