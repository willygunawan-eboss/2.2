# System Health

## Overview
Menu `ERP System Health` menampilkan hasil diagnostic core infrastructure dan modul ERP secara real-time.

## Metrik Diagnostic
- Database
- API Server
- Migration
- Seeder
- Reference Engine
- RBAC
- Dan berbagai master modul lainnya.

## Indikator
- **Pass** (Hijau): Kondisi optimal
- **Warning** (Kuning): Ada limitasi / peringatan yang tidak kritikal
- **Error** (Merah): Sistem mengalami error atau belum terkonfigurasi.
