# ICHANGEBOSS ERP - Enterprise Bootstrap Framework

## Latar Belakang
Sistem ERP skala Enterprise tidak dapat langsung digunakan setelah instalasi. Harus melalui fase "Initial Setup" untuk mencegah error "Foreign Key Constraint" akibat Master Data yang kosong.

## Fitur Utama
1. **Full Screen Wizard**: Mengambil alih tampilan jika `systemReady === false`.
2. **Progress Tracking**: Menampilkan persentase kelengkapan.
3. **10-Step Setup**: Dari Company Profile hingga Employee 1st.
4. **Validation Layer**: Memastikan urutan data master diisi dengan benar.
