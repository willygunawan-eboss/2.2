# Team Module (Enterprise)

## Overview
The Team Module handles enterprise team hierarchy, falling under Section.

## Features
- Master-Detail interface
- Soft Delete & Restore
- Real-time Audit Logging
- Filtering by Company, Branch, Division, Department, Section
- Validation with Zod

## Dependencies
- Section
- Department
- Division
- Branch
- Company

## RBAC
Resource: `team`
Actions: `read`, `write`, `delete`
