# Employee Domain Model

## Core Segments
1. **Personal Identity**: UUID, Employee Number, NIK, Name, Demographics.
2. **Employment Information**: Status, Type, Dates (Hire, Join, Contract).
3. **Organization Assignment**: Company, Branch, Division, Department, Section, Team, Position, Job Grade.
4. **Communication**: Corporate/Personal Email, Phones, Emergency Contacts.
5. **Audit**: Status (Active/Inactive), Timestamps, User references for Create/Update/Delete.
