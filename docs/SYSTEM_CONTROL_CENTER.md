# Enterprise Control Center

## Overview
The Enterprise Control Center is the centralized dashboard for configuring the ERP system. It replaces the traditional left-nav Settings view with a unified, dashboard-driven experience.

## Module Groupings
Settings and configuration items are categorized into semantic groups, surfaced as interactive cards:

1. **Organization**: Company, Branch, Division, Department, Section, Team, Position, Job Grade.
2. **Human Resources**: Working Calendar, Holiday, Shift, Employment Type, Leave Policy, Payroll Policy.
3. **Master Data**: Banks, Currencies, Locations, Cost Centers.
4. **Workflow**: Approval Matrix, Numbering Rules, Notification.
5. **Security**: Users, Roles, Permissions, RBAC Cache, Audit Log.
6. **Infrastructure**: ERP Setup Center, Backup & Restore, Database Health, License.
7. **Integration**: API Keys, Webhooks, SSO Configuration.
8. **Monitoring**: System Health, Performance Metrics, Error Logs.
9. **AI**: AI Models, Automation Rules, AI Activity Logs.
10. **System**: General Settings, Branding & Theme, Localization.

## User Flow
- Users navigate to the Control Center to view the status and progress of all modules at a glance.
- Clicking on a module card opens a dedicated layout (category view) containing specific settings elements.
- The `SetupCenterView` is also accessible under Infrastructure for a structured approach to system onboarding and readiness checks.
